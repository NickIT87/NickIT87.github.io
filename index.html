project description

Features - cipher: cyber cipher module: web-interface that implements a symmetric encryption system for text files and A message based on a self - contained security protocol and mutual trust users.

Download and read UserGuide.txt

for the correct application use the google chrome, opera browser. Mozilla browser not supported.

Features - cipher: The Gronfeld cipher is a polyalphabetic substitution cipher created by Count Gronsweld (head of the first decipherment service of Germany) in the 17th century. Cipher can be considered an improvement in the cipher of Caesar (reliability) and Vigenera / Beaufort (speed).

The application only accepts text, special characters and punctuation marks are not processed.

Features - bot: simple chat-bot that implements the scheme of the system of questions and answers. It offers the possibility of solving the problem of faulty equipment that was damaged as a result of misuse. Similarly, under certain conditions the bot responds with a random winged expression in English and German.
